Matematicando – Aplicativo Educacional

Esta é a primeira versão do Matematicando, um aplicativo com interface gráfica desenvolvido em Python, utilizando as bibliotecas Tkinter e CustomTkinter. Trata-se de uma ferramenta educacional complementar, voltada para o aprendizado e a prática das quatro operações básicas da matemática: adição, subtração, multiplicação e divisão.

O Matematicando apresenta uma interface intuitiva e acessível, projetada para usuários de todas as idades, com diversos níveis de dificuldade. Foi desenvolvido no âmbito de um projeto de pesquisa e inovação tecnológica, focado na criação de recursos educacionais digitais, visando integrar conhecimentos em programação e no ensino-aprendizagem da matemática.

Espera-se que o uso do Matematicando contribua para o desenvolvimento de habilidades matemáticas em consonância com a Base Nacional Comum Curricular (BNCC) para o Ensino Fundamental, especialmente nos seguintes componentes: EF03MA06, EF03MA07, EF03MA08, EF05MA07 e EF05MA08.

Este aplicativo está disponível para sistemas operacionais Windows.
